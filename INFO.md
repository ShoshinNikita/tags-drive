# Additional info

## Meta information

The main file is `files.json`. It keeps information about files and their tags.

**Basic structure:**

```json
{
  "filename": {
    "filename": "filename",
    "size": 123,
    "tags": [],
    "add_time": "2018-08-19T17:15:41.7045266+03:00"
  }
}
```

## Files

All files are kept in `data` folder.
